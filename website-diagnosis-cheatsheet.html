Sabse Fast Check — 10 seconds

Mobile data ON karo → same URL kholo → Khul gaya? WiFi/ISP issue hai | Nahi khula? Website ka server hi down hai

3 Steps — Ek ke baad ek chalao
1
DNS Check
Domain → IP resolve ho raha hai?
C:\> nslookup beeindia.gov.in
✅
IP address mila (164.x.x.x)
DNS bilkul theek hai → Step 2 jao
❌
"can't find" ya "NXDOMAIN"
DNS server problem → Control Panel > DNS: 8.8.8.8 set karo
⚠️
"Server: Unknown"
Sirf router DNS use ho raha — normal hai, chinta mat karo
2
Network / Routing Check
Kahan pe ruk raha hai connection?
C:\> tracert beeindia.gov.in
✅
Trace complete — last hop = server IP
Network theek hai, server tak pahunche → Step 3 jao
❌
Beech mein "* * * Request timed out"
Us hop pe ISP ya firewall block kar raha — VPN try karo
⚠️
Last hop pe * * * aaya
Server ICMP ping block karta hai — curl se confirm karo
3
Port / Web Service Check
HTTP/HTTPS respond kar raha hai?
C:\> curl -v https://site.com
C:\> curl -v http://site.com
✅
HTTP 200 OK
Server theek — browser cache clear karo, Ctrl+Shift+Del
❌
port 443 failed: Timed out
HTTPS service crash/down — sirf server admin fix kar sakta
⚠️
HTTP 301/302 → redirect
Domain change ya http→https redirect — follow the redirect
Decision Flow — Result dekho aur seedha answer pao
Q1nslookup ne IP diya?
✅ Haan DNS theek → tracert chalao
❌ Nahi DNS broken → 8.8.8.8 set karo ya ISP call
Q2tracert complete hua?
✅ Haan (last hop = server) Network OK → curl se port check karo
❌ Beech mein * * * ISP/Firewall block → VPN try karo
Q3curl port 443 timed out?
✅ 200 OK aaya Server theek — browser ya cache issue
❌ Timed out 🎯 CONFIRMED: Web service server pe down hai
⚠️ Connection refused Firewall ne port block kiya — server side issue
Q4Mobile data pe khul raha?
✅ Haan, mobile pe khula Tera WiFi/ISP problem hai — router restart karo
❌ Mobile pe bhi nahi khula Website hi down hai — NIC/BEE ka server issue
